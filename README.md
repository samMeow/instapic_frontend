# InstaPic Frontend
Allow user to create post with videos or image. Hosted: https://sammeow-instapic-frontend.herokuapp.com/

## Getting Started
Require: yarn, node > 12

1. Run Application
```sh
yarn start
```
2. Testing / Linting
```sh
# testing
yarn test
# linting
yarn lint
```
3. Integration Test (start app first)
```
yarn test:integration
```
